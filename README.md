# NexTask AI — Ultimate Productivity Workspace

NexTask AI aik high-performance Task Management application hai jo **C++ (Raylib GUI)** aur **Python AI Architecture** ke behtareen integration ke sath bani hai. Is mein user-specific management ke sath advance AI features shamil hain.

---

## 🛠 Core Features

* **Google Sign-In Gateway:** Secure authentication loopback web server bridge ke zariye (Real browser handshake).
* **AI Context Chatbot:** **Groq Cloud (Llama 3.1)** ke zariye aap ke real-time tasks database ka dynamic analysis aur Urdu/English blend mein smart advice.
* **Voice AI Assistant:** Hands-free task creation aur NLP date detection via speech-to-text bridge.
* **Advanced Sorting Engine:** Permanent database rearrangement via highly optimized **Merge Sort** algorithm on double-linked lists.
* **7-Day Dynamic Themes:** Monday se Sunday tak har din ka automatic alag graphical color matrix environment.
* **Smooth Navigation:** Full scrolling panels for dashboard, timeline modifiers (+/- progress increments), calendar views, and complete undo/redo state history (`Ctrl + Z`).
